Question 6 :

Whats the difference between window vs document?

        Window
        The window is the first thing that gets loaded into the browser. This window object has the majority of the 
        properties like length, innerWidth, innerHeight, name, if it has been closed, its parents, and more.

        Document
        The document object is your html will be loaded into the browser. The document actually gets loaded inside the window 
        object and has properties available to it like title, URL, cookie, etc. That means if you want to access a 
        property for the window it is window.property, if it is document it is window.document.property which is also 
        available in short as document.property.
